# How to use it #
Just put this in your `Gemfile`:
    
    gem "infopark_rails_connector_meta"
    gem "infopark_reactor_migrations"
    gem "infopark_reactor"

# How to get help #
Read the `README` files for the gems, if that doesn't help - ask.
